# Возможности портала

`Orienteering Portal` — локальный веб-портал для разбора тренировок и стартов по спортивному ориентированию.

## Что умеет портал

- Вести список тренировок и стартов в одном месте.
- Добавлять новую тренировку через пошаговый мастер.
- Редактировать уже сохранённую тренировку без создания дублей.
- Загружать карту тренировки.
- Привязывать карту к координатам по опорным точкам.
- Загружать GPX-трек.
- Сохранять дистанцию, тип тренировки, место и заметки.

## Работа с треком

- Открывать тренировку в режиме плеера.
- Проигрывать движение по дистанции на карте.
- Менять скорость воспроизведения.
- Перематывать трек ползунком.
- Смотреть текущий темп по ходу движения.
- Обрезать начало и конец трека.
- Отменять обрезку.
- Сохранять обновлённый трек после правок.

## Анализ сплитов

- Автоматически рассчитывать сплиты по ближайшим КП.
- Показывать время на каждом перегоне.
- Показывать длину этапа и темп.
- Подсвечивать быстрые и проблемные этапы.
- Фильтровать только проблемные сплиты.
- Открывать детальный анализ отдельного сплита.

## Работа с протоколами

- Импортировать протокол соревнований по URL.
- Выбирать свою группу и свою строку в протоколе.
- Сохранять протокол в портал.
- Открывать уже сохранённый протокол из списка тренировок.
- Смотреть отдельный список всех сохранённых протоколов.
- Удалять ненужные протоколы.

## Разбор результата

- Сравнивать свои сплиты с лидером группы.
- Видеть отставание на каждом этапе.
- Получать цветовую маркировку хороших, средних и проблемных этапов.
- Видеть суммарный проигрыш на проблемных этапах.
- Открывать связанный плеер тренировки прямо из протокола.

## Для чего это полезно

- Разбирать ошибки после старта.
- Сопоставлять трек с официальным протоколом.
- Находить самые дорогие по времени этапы.
- Сравнивать темп, выбор пути и потери по дистанции.
- Хранить тренировочную историю в одном локальном инструменте.
